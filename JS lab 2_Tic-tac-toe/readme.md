# Tic-Tac-Toe Game

A simple, interactive Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Description

This is a classic Tic-Tac-Toe game where two players can play against each other. The game features a clean, responsive design and includes scoring, win detection, and a reset functionality.

## Features

- Two-player gameplay (X and O)
- Real-time win detection
- Score tracking for each player
- Draw detection
- Reset game functionality
- Responsive design
- Smooth animations

## File Structure

- `main.html` - The main game file containing HTML, CSS, and JavaScript
- `readme.md` - This README file

## Installation and Usage

No installation required! Simply clone the repository and open `main.html` in any modern web browser.

```bash
git clone <repository-url>
cd <repository-directory>/JS lab 2
open main.html
```
