# Tic-Tac-Toe Game

A browser-based Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## How to Play

1. Open `main.html` in your web browser
2. Players X and O take turns clicking on empty cells
3. First player to get 3 in a row (horizontally, vertically, or diagonally) wins!
4. Click "Reset Game" to start a new game

## Features

- Interactive 3x3 game board
- Turn-based gameplay (X and O)
- Win detection with highlighted winning cells
- Draw detection
- Score counter for X wins, O wins, and draws
- Smooth animations and hover effects
- Responsive design

## Technologies Used

- HTML5
- CSS3 (with animations)
- Vanilla JavaScript (DOM manipulation and event listeners)

## Learning Objectives

This project demonstrates:
- DOM manipulation
- Event listeners
- Game state management
- CSS animations and transitions
- Responsive web design
